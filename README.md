# Rock Paper Scissors 🎮

First Streamlit project I built while learning Python.
Wanted something interactive so this felt like a fun starting point.

## What it does
- Play Rock Paper Scissors against the computer
- Computer picks randomly every round
- Tracks score as you play
- Reset button to start fresh

## What I learned
- How Streamlit works for building quick UI with Python
- Using session_state to keep data between clicks
- Refactored win logic to use a dictionary instead of big if elif block

## Tech used
Python, Streamlit

## How to run
pip install -r requirements.txt
streamlit run app.py
